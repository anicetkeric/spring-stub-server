# spring-stub-server

Standalone stub server using Spring Cloud Contract WireMock

## Tech Stack

**Spring-boot**, **maven**, **java17**, **docker**, **WireMock**

## Run Locally

Clone the project

```bash
  git clone https://github.com/anicetkeric/spring-stub-server.git
```
